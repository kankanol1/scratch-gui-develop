# 目录

## 精简版
```text
|-- scratch-gui
    |-- README.md
    |-- webpack.config.js
    |-- build
    |   |-- blocks-only.html      # blocks 编辑器
    |   |-- compatibility-testing.html
    |   |-- extension-worker.js
    |   |-- index.html            # scratch 编辑器
    |   |-- lib.min.js            # scratch 核心
    |   |-- player.html           # scratch 播放器
    |   |-- chunks                # Scratch核心加载器
    |   |-- static                # 静态文件
    |-- src
    |   |-- components            # UI组件
    |   |-- containers            # 容器组件，承载容器组价业务逻辑
    |   |-- css                   # 全局通用CSS
    |   |-- examples              # 集成测试用例
    |   |   |-- extensions        # 拓展案例
    |   |-- lib                   # 插件及高阶组件
    |   |   |-- audio             # 声音插件
    |   |   |-- backpack          # 背包插件
    |   |   |-- default-project   # 默认项目
    |   |   |-- libraries         # 素材库相关
    |   |   |-- video             # 视频模块
    |   |-- playground            # 翻译后页面的模板
    |   |-- reducers              # 全局状态控制
    |-- static                    # 静态文件
    |   |-- favicon.ico
    |-- test                      # 测试用例
    |-- translations              #翻译库
```
